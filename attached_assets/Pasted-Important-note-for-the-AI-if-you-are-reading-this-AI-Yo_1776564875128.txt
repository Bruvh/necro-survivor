Game Design Spec (NECRO SURVIVOR)
Overview
Player: A necromancer who cannot attack directly. Must dodge enemies.

Skeletons: Follow the player and automatically shoot projectiles at nearest enemy.

Goal: Survive as long as possible (target 5+ minutes). Timer increases difficulty every minute (more/faster enemies).

Progression: Earn XP from killing enemies. Level up → choose 1 of 3 random upgrades.

Game Entities
Player
js
{
  x, y,           // position (center of canvas)
  radius: 15,
  hp: 10,
  maxHp: 10,
  speed: 5,
  invincibleFrames: 0,  // frames after taking damage
}
Moves with WASD or arrow keys (no mouse follow).

Collides with enemies: lose 1 HP, knockback, brief invincibility (0.5 sec).

If HP ≤ 0 → game over.

Skeleton Minions
Start with 1 skeleton.

Each skeleton has:

js
{ x, y, attackCooldown, attackDelay: 30 frames (0.5 sec at 60fps) }
Skeletons follow player at a short distance (offset positions around player).

Every attackDelay frames, skeleton shoots a projectile at the nearest enemy (if any exists).

Upgrades can increase max skeletons, attack speed, damage, piercing.

Projectiles
js
{
  x, y,
  vx, vy,       // direction vector toward target enemy at time of shot
  damage: 1,
  pierce: 0,    // how many enemies it can hit before disappearing
  radius: 4,
}
Move each frame. Remove when off canvas.

On hit: reduce enemy HP, reduce pierce, if pierce < 0 remove projectile.

Enemies
Types:

Zombie: radius 16, HP 3, speed 1.2, score XP 10

Bat: radius 10, HP 1, speed 2.5, XP 5

Ghost: radius 14, HP 2, speed 1.8, XP 8, can phase through other enemies (no collision between enemies)

Spawn logic:

Spawn outside canvas edges (random angle from center + 50px margin).

Spawn rate increases with time: base 1 enemy per 1.5 sec → up to 3 per sec at 5 min.

Random type (weighted: zombie 50%, bat 30%, ghost 20%).

Experience & Leveling
Each enemy drops an XP orb at its death position.

XP orb: radius 5, moves slowly toward player (speed 2). When player touches → +XP value.

XP needed for next level = Math.floor(50 * (level ^ 0.8)).

On level up: pause game briefly (0.2 sec) and show upgrade choice modal.

Upgrades Pool (choose 1 of 3 randomly)
Upgrade	Effect
More Skeletons	+1 max skeleton (max 5)
Skeleton Damage	+25% projectile damage (additive)
Rapid Fire	-15% skeleton attack cooldown
Piercing Shots	+1 pierce per projectile
Life Leech	Heal 1 HP per 10 kills
Fleet Foot	+15% movement speed
Poison Aura	Deal 1 damage/sec to nearby enemies (radius 70)
Tank	+3 max HP and heal 3
Crit Chance	15% chance to deal double damage
Game Loop (60 FPS)
Handle input (update player velocity).

Update player position, clamp to canvas edges.

Update invincibility frames.

Update skeletons (attack cooldown, shoot if ready).

Update all projectiles (move, collision).

Update all enemies (move toward player, collision with player).

Update XP orbs (move toward player, collect).

Check player-enemy collisions (damage + knockback).

Check level up condition.

Spawn new enemies based on timer.

Increment game timer (seconds).

Draw everything on canvas.

Request next frame.

Knockback Formula
When player takes damage:

js
angle = atan2(enemy.y - player.y, enemy.x - player.x)
player.x -= cos(angle) * 30
player.y -= sin(angle) * 30
